These directories & files provide examples of Go code
implementing a network-based asynchronous buffer using Go RPC

rpc/bufi/bufi.go:
  Slightly modified version of sequential buffer from class 04

rpc/dserver/dserver.go:
  Slightly modified version of asynchronous buffer from class 05

rpc/rpclib/rpclib.go:
  Implements server and client functions for buffer

rpc/server/server.go:
  Server application

rpc/client/client.go:
  Client application
